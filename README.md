# Bank-Loan-Analysis
This project involves analyzing a bank loan dataset and understanding the terminologies used in loan processing. The main objective is to identify key factors affecting loan approval, customer risk levels, and repayment behaviour using essential loan fields such as annual income, DTI, loan grade, employment details, and loan status.

## Approach / Methodology
## 1. Data Cleaning
Removed missing values
Standardized date formats
Cleaned employment title and length fields
Validated numerical columns (DTI, Annual Income, Interest Rate)

## 2. Data Transformation
Derived loan age
Categorized borrowers by income levels
Created risk buckets based on Grade/Sub-Grade

## 3. Exploratory Data Analysis (EDA)
Loan distribution by category
DTI vs Default probability
Income vs Loan Amount
Employment length vs repayment behaviour
